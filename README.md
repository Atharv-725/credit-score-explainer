# 💳 Credit Risk Predictor

An interactive web app that predicts loan default risk based on 
financial details using Machine Learning.

## 🚀 Live Demo
Run locally with:
```bash
streamlit run app.py
```

## 🛠️ Tech Stack
Python | Streamlit | Scikit-learn | Pandas | NumPy

## 📊 Features
- Interactive sliders for financial inputs
- Real-time loan default prediction
- Risk probability breakdown
- Input summary table

## 🧠 Model
- Random Forest Classifier
- Trained on synthetic financial dataset
- Features: Income, Age, Loan Amount, Credit Score,
  Employment Years, Debt Ratio, Credit Lines, Payment History

## ⚙️ Installation
```bash
pip install streamlit scikit-learn pandas numpy
streamlit run app.py
```
